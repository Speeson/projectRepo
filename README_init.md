# Hackatón FP RepoProjects - Starter Repo

## Estructura
- frontend/ (Angular)
- backend/ (Node + Express + TypeScript)
- db/ (MySQL schema.sql y seed.sql)
- docs/ (backlog y documentación)

## Requisitos
- Node.js LTS
- Angular CLI
- MySQL Server
- VS Code
- Git

## Base de datos
1. Ejecuta db/schema.sql
2. Ejecuta db/seed.sql (sustituye HASH_BCRYPT_123456 por un hash real durante el hackatón)

## Backend
- Copia backend/.env.example a backend/.env y ajusta valores
- En el hackatón se inicializa el proyecto Express TS

## Frontend
- En el hackatón se crea el proyecto Angular en frontend/
